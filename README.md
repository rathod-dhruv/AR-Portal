# AR-Portal

Augmented Reality Portal Experience - An immersive AR app that creates virtual portals to alternate worlds.

## Overview

AR-Portal is an augmented reality application that creates immersive portal experiences on mobile devices. Users can open virtual doorways in their physical environment and step through to explore alternate worlds and environments in real-time.

## Features

- **Portal Creation**: Place interactive portals in AR environment
- **World Transitions**: Seamless switching between real and virtual worlds
- **Environmental Mapping**: Portals adapt to room layout and lighting
- **Interactive Elements**: Explore and interact with virtual environments
- **Multi-platform**: Works on both Android and iOS devices
- **Performance Optimized**: Smooth 60+ FPS experience

## Tech Stack

- **Engine**: Unity 2019.4 LTS
- **AR Framework**: AR Foundation
- **AR Libraries**: ARCore (Android) / ARKit (iOS)
- **Language**: C#
- **Platform**: Mobile (Android & iOS)
- **UI**: Unity Asset Store components

## Installation & Setup

### Prerequisites

- Unity 2019.4 LTS
- AR Foundation
- ARCore (Android) or ARKit (iOS) SDK
- Mobile device with AR support

### Build Instructions

1. Open project in Unity
2. Ensure AR Foundation is installed
3. Configure build settings
4. Build for target platform
5. Deploy to device

## Usage

1. Launch application on AR-capable device
2. Allow camera and location permissions
3. Point device at flat surface (wall, floor)
4. Tap to place portal
5. Walk through portal to enter virtual world
6. Explore and interact with environment
7. Return through portal to exit
